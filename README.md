# Bookshop - Ejercicios Finales DWEC

Este repositorio contiene una colección de ejercicios prácticos para la asignatura de Desarrollo Web en Entorno Cliente (DWEC). El objetivo es guiarte paso a paso en la creación de una aplicación de gestión de libros, abordando tanto el backend como diferentes enfoques de frontend (vanilla, Bootstrap y React).

## Estructura del repositorio

- **README-EXERCISES.md**: Documento con los enunciados y las instrucciones detalladas para cada ejercicio. Aquí encontrarás el orden recomendado y las tareas a realizar en cada etapa.
- **README-SOLUTIONS.md**: Documento con las soluciones propuestas para cada ejercicio. Úsalo como referencia o para comparar con tu propia implementación.
- **back/**: Código fuente del backend (Node.js + Express).
- **front-cart/**, **front-crud/**: Ejercicios de frontend con JavaScript vanilla.
- **front-cart-bootstrap/**, **front-crud-bootstrap/**: Ejercicios de frontend usando Bootstrap.
- **front-cart-react/**, **front-crud-react/**, **front-all-react/**: Ejercicios de frontend usando React.

## ¿Cómo usar este repositorio?

1. Lee primero el archivo **README-EXERCISES.md** para conocer el orden y los requisitos de cada ejercicio.
2. Realiza los ejercicios siguiendo las instrucciones.
3. Consulta **README-SOLUTIONS.md** solo para comparar o resolver dudas, no como primer recurso.
4. Puedes ejecutar y modificar el código de cada carpeta según las instrucciones de cada ejercicio.

## Requisitos previos

- Node.js y npm instalados
- Navegador web moderno

## Notas

- El backend debe estar en funcionamiento para probar los frontends.
- Las soluciones son orientativas y pueden variar en implementación.
- Contenidos adpatados para el módulo de DWEC del CFGS DAW
